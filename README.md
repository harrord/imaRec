# 人生记录器 (imaRec)

**人生记录器 (imaRec)** 是一款为腾讯 ima 开发的安卓应用。它的核心工作流是：在本地进行录音，随后自动上传至 ima 进行处理与存档。

本 APP 可作为最近几年爆火的 AI 硬件产品——录音卡片这个品类的平替。不仅对于那些喜欢手机套壳的人友好，而且录音卡片当中会议总结、纪要、翻译、区分说话人、脑图等等功能也都可以实现，并且完全免费。单次录音上限两小时，总量则没有上限。

在 AI 时代，个人数据变得前所未有地重要。时间会流逝，记忆会变形，而记录是为了让记忆转化为数据，永久地留存下来。我们致力于将所有的个人数据留存，以实现某种意义上的“赛博永生”。

本应用的最初设计目标是 **24 小时全天候录取你身边值得被记录下的声音**，尽可能为你留下身边那些值得被记录的声音，而无关的噪音则不予记录。

请移步 release 页面下载 apk 后安装。

## ✨ 核心理念与智能规则

为了实现全天候、智能化的声音记录，应用内置了多种中断与重启规则：
- **定时中断**：例如在晚上睡觉时，可以选择设定定时自动中断录音。
- **环境自适应**：当外部声音分贝过小（说明正处于极其安静的环境）时，应用会自动中断录音以节省资源。
- **智能唤醒**：当你重新开始步行、或者到达某个特定地点后，录音将会重新启动。
- **地理触发**：可以预设地点，到达指定地点范围时自动开启新录音或切出新的录音片段。
- **闪念胶囊**：可以扫描系统日历，把特定时间锚点的新日程自动转成 ima 笔记。

**🔋 耗电表现**
取决于您对自动启停录音的设置。

## ✨ 功能特性

目前已实现以下核心功能：
- **本地与云端联动**：录制本地声音并上传至腾讯 ima 服务器。
- **录音管理**：主页面直观展示所有录音文件列表，方便您浏览和管理。
- **一键录音**：采用底部中央大型按钮设计，支持一键快速开始或结束录音。
- **高品质存储**：录音文件默认保存为高质量的 `.m4a` 格式，兼顾音质和存储空间。
- **后台稳定录音**：录音期间会显示前台服务通知及锁屏通知，确保应用退至后台时录音不被中断。
- **通知栏便捷控制**：支持直接在通知栏内暂停、继续、切换分组、记录灵感。
- **ima 文件夹联动**：在同一个 ima 知识库内管理多个文件夹，录音会上传到当前选中的文件夹。
- **灵感记录**：可将临时灵感单独保存到指定灵感文件夹，不影响默认上传文件夹。
- **地理触发录音**：到达预设地点时自动开始新录音或切出新片段，离开范围时可按配置停止录音。
- **闪念胶囊**：定时扫描系统日历，把符合规则的新日程自动创建为 ima 文本笔记。

## 🔒 锁屏卡片及通知栏快捷操作

本应用提供了强大的前台通知及锁屏控制卡片，无需解锁屏幕，即可直接在锁屏界面或下拉通知栏对录音进行精细化管理。当前锁屏卡片提供三个快捷按钮：**分组**、**灵感**、**暂停/继续**。

### 1. 📂 分组按钮
*   **用途**：在主页已展开的 ima 文件夹中切换当前录音片段的保存目标。
*   **启用条件**：主页已展开的文件夹数量 $\ge 2$。
*   **录音状态下点击**：循环切换到下一个文件夹，并启动 5 秒倒计时。
*   **5 秒内连续点击**：继续循环切换文件夹，并重新开始 5 秒倒计时。
*   **5 秒内无再次点击**：倒计时结束后，自动执行分段：当前片段保存并上传到切换后的文件夹，同时立即开启新片段继续录音。
*   **反馈提示**：通知卡片会显示当前选中的文件夹和倒计时提示，锁屏状态下也可见。

### 2. 💡 灵感按钮
*   **用途**：把临时想法、突发念头、备忘内容单独保存到“灵感文件夹”。
*   **启用条件**：已在 ima 配置中选择灵感目标文件夹。未配置时按钮置灰不可点击。
*   **普通录音状态下点击**：立即保存当前普通录音片段，并进入灵感记录模式。
*   **灵感记录模式下再次点击**：保存当前灵感片段到灵感文件夹，并回到普通录音模式。
*   **息屏自动保存**：如果正在记录灵感，按电源键息屏时会自动保存灵感，避免临时想法丢失。
*   **不影响默认文件夹**：灵感文件夹只是灵感记录的目标文件夹，不会变成后续普通录音的默认上传文件夹。

### 3. ⏸️ 暂停/继续按钮
*   **单次/首次点击**：进入“暂停选择窗口”（开启 5 秒倒计时，倒计时期间录音继续进行）。通知栏和锁屏卡片反馈行会显示“5 秒后暂停”提示，暂停按钮文案显示“暂停”。
*   **5 秒内连续点击**：循环切换不同的暂停模式，每次点击都会重新开始 5 秒倒计时。支持 4 档循环：
    *   **第 1 次点击（或 4n+1 次）**：一直暂停（倒计时结束后需要手动恢复）。
    *   **第 2 次点击（或 4n+2 次）**：定时暂停 X 分钟（例如 5 分钟，具体由设置中的 `pauseMinutesX` 决定）。
    *   **第 3 次点击（或 4n+3 次）**：定时暂停 Y 分钟（例如 10 分钟，由 `pauseMinutesY` 决定）。
    *   **第 4 次点击（或 4n 次）**：定时暂停 Z 分钟（例如 30 分钟，由 `pauseMinutesZ` 决定）。
*   **5 秒内无再次点击**：倒计时结束后真正执行暂停，并根据选择的模式：
    *   若为“一直暂停”：需手动再次点击“继续”恢复录音。
    *   若为“定时暂停”：通知栏和锁屏卡片将显示剩余时间倒计时（如“暂停剩余 N 分钟”），倒计时归零时自动恢复录音。
*   **暂停状态下点击**：直接恢复录音，返回录音状态。

## 📁 ima 知识库与文件夹

**人生记录器** 现在采用“一个 ima 知识库 + 多个文件夹”的组织方式：

- 你需要先选择一个目标 ima 知识库。
- APP 会在这个知识库内读取文件夹列表。
- 主页上展开的 Tab 对应的是该知识库下的文件夹。
- 普通录音会上传到当前选中的文件夹。
- 锁屏“分组”按钮会在已展开的文件夹之间循环切换，并在倒计时结束后把当前片段保存到切换后的文件夹。
- 灵感记录有独立的灵感目标文件夹，不会改变普通录音的默认上传目标。

这种方式更适合把同一个人的长期记录沉淀在一个知识库中，再通过文件夹区分不同场景，例如“日常记录”“会议”“课程”“灵感”等。

## 📍 地理触发录音

地理触发录音用于记录“到达某个地点时发生的声音”。你可以提前配置一个或多个预设地点，并设置触发半径。

支持的能力包括：
- **到达地点自动录音**：进入预设地点范围后，APP 会自动开始新录音；如果当时正在录音，则会强制切出一个新片段。
- **地点备注写入文件名**：触发录音时，文件名会带上地点备注，方便之后在 ima 中检索。
- **离开地点自动停止**：可选择离开预设范围后自动停止录音。
- **后台定时扫描**：地理触发服务独立运行，会按设定间隔检查当前位置。
- **震动与提示**：触发成功时会震动并显示提示，便于确认已经开始记录。

添加地点时，除了使用当前位置，也可以从照片中读取 GPS 信息：
- 选择一张带有地理信息的照片；
- APP 会读取照片 EXIF 中的经纬度；
- 可将该坐标保存为一个地理触发地点。

## 🗓️ 闪念胶囊

“闪念胶囊”用于把系统日历中的新日程自动变成 ima 笔记，适合用来捕捉临时备忘、稍后处理事项、任务胶囊等文字型信息。

工作方式：
- APP 会定时扫描系统日历中未来 7 天的日程。
- 你可以设置一个“锚点小时”，例如只处理凌晨 3 点开始的日程。
- 符合锚点小时且尚未处理过的新日程，会被创建为 ima 文本笔记。
- 日程标题会作为笔记标题；日程标题和描述会组成笔记正文。
- 笔记创建成功后，APP 会记录该日程已处理，避免重复上传。

为了避免首次开启时误上传历史日程，闪念胶囊加入了**基线快照**机制：
- 第一次开启功能时，会把当时未来 7 天内已有的日程全部标记为已处理；
- 首次扫描不会上传这些旧日程；
- 之后只处理基线快照之后新增的日程。

## ⚙️ 联动 ima 配置

**人生记录器** 需要和腾讯的 ima 联动配合使用。ima 提供了移动端、桌面端以及网页端等多种使用方式。

为了让本 APP 能够自动将录音上传至服务器，请按照以下步骤获取和使用相关配置：
1. 访问 ima 开发者接口页面：[https://ima.qq.com/agent-interface](https://ima.qq.com/agent-interface)
2. 用您自己的账户登录该网页。
3. 在页面中获取您的 **API key** 和 **Client ID**。
4. 将获取到的 API key 和 Client ID 配置到“人生记录器” APP 中。
5. 选择目标知识库，并在该知识库中添加需要使用的文件夹，即可实现录音自动上传与联动。

### 4. 权限说明
为了让全天候后台录音应用正常运行，首次打开及使用时会请求必要的权限：
- **麦克风权限 (`RECORD_AUDIO`)**：必须的权限，用于采集环境声音。
- **通知权限 (`POST_NOTIFICATIONS`)**（对于 Android 13 及以上）：用于显示后台录音的控制通知栏。
- **忽略电池优化 / 后台耗电权限**：**（极其重要）** 为了防止系统杀掉后台进程，确保 24 小时记录不被中断，您必须在系统设置中允许应用后台耗电或将其加入无限制白名单。
- **定位权限 (`ACCESS_FINE_LOCATION` / `ACCESS_COARSE_LOCATION` / `ACCESS_BACKGROUND_LOCATION`)**：用于地理触发录音，在后台判断是否到达预设地点。
- **照片 / 媒体读取权限**：用于从照片 EXIF 中读取 GPS 信息，帮助创建地理触发地点。
- **日历读取权限 (`READ_CALENDAR`)**：用于闪念胶囊扫描系统日历并创建 ima 笔记。
